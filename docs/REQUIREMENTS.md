# 需求文档

> 本文档定义 KNOT 系统的功能需求与非功能需求。

## 一、功能需求

### F1. 智能任务规划
- **F1.1** 支持自然语言指令输入，自动识别用户意图
- **F1.2** 将复杂任务拆解为可执行的子任务序列，生成图结构工作流
- **F1.3** 支持用户对规划结果进行可视化预览与人工干预调整
- **F1.4** 支持工作流模板的保存、导入与复用

### F2. 知识增强
- **F2.1** 集成向量数据库，支持语义化知识检索
- **F2.2** 支持结构化知识库与非结构化文档的双路检索
- **F2.3** 支持多轮对话中的上下文记忆与连续历史追踪
- **F2.4** 支持任务执行过程中的动态知识注入

### F3. 多智能体协同
- **F3.1** 支持多智能体在统一编排框架下的分工协作
- **F3.2** 支持智能体角色定义（规划者、执行者、校验者、总结者）
- **F3.3** 支持智能体间的状态共享与数据流转
- **F3.4** 支持智能体执行的暂停、恢复与手动干预

### F4. 工作流执行
- **F4.1** 支持串行、并行、条件分支、循环等多种执行模式
- **F4.2** 具备任务依赖关系解析与拓扑排序能力
- **F4.3** 提供任务执行的超时控制与重试机制
- **F4.4** 支持工作流执行的暂停、恢复与终止

### F5. 执行适配
- **F5.1** 支持 RESTful API 调用
- **F5.2** 支持数据库操作（SQL/NoSQL）
- **F5.3** 支持第三方服务集成（通过插件机制）
- **F5.4** 支持本地命令/脚本执行

### F6. 可观测性
- **F6.1** 全链路任务执行追踪与日志记录
- **F6.2** 实时展示各子任务的执行状态、耗时及输出
- **F6.3** 异常检测与故障节点定位
- **F6.4** 性能监控与告警

### F7. 用户界面（GUI 可视化编排）

**工作流可视化编辑器**
- **F7.1** 提供图形化拖拽式工作流编辑界面，支持从组件面板拖拽节点到画布
- **F7.2** 画布支持缩放、平移、对齐辅助线、网格吸附等专业编辑体验
- **F7.3** 节点间连线支持拖拽操作，自动识别兼容性并提示连接约束
- **F7.4** 编辑过程中实时预览工作流拓扑结构与完整执行路径

**运行监控看板**
- **F7.5** 工作流运行时实时展示各节点执行状态（待执行/运行中/成功/失败）
- **F7.6** 节点详情面板：点击节点展示输入/输出数据、耗时、日志与错误信息
- **F7.7** 全链路执行时间轴视图，直观展示串行等待与并行重叠关系
- **F7.8** 异常节点高亮标记与故障根因定位建议

**知识库管理界面**
- **F7.9** 知识库内容管理与配置界面，支持文档上传、分段预览与检索测试
- **F7.10** 知识检索效果的在线调试与对比测试

**系统管理**
- **F7.11** 系统配置与管理界面，包括用户权限、API Key、日志查询等
- **F7.12** 深色模式与响应式布局支持

## 二、非功能需求

### N1. 性能
- **N1.1** 百级节点工作流编排延迟 < 500ms
- **N1.2** 知识检索响应时间 < 200ms（P99）
- **N1.3** 系统支持 100+ 并发任务执行

### N2. 可用性
- **N2.1** 系统可用性 ≥ 99.9%
- **N2.2** 支持优雅降级，核心功能不因单点故障完全不可用

### N3. 安全性
- **N3.1** 支持基于角色的访问控制（RBAC）
- **N3.2** API 通信全程 TLS 加密
- **N3.3** 敏感配置信息加密存储

### N4. 可扩展性
- **N4.1** 工具/API 集成采用插件化架构，即插即用
- **N4.2** 智能体类型可扩展
- **N4.3** 知识库支持水平扩展

### N5. 可维护性
- **N5.1** 模块化设计，各层职责清晰
- **N5.2** 完整的 API 文档与开发者指南
- **N5.3** 单元测试覆盖率 ≥ 80%

## 三、技术栈建议

| 层次 | 技术选型 |
|------|----------|
| 层次 | 技术选型 |
|------|----------|
| 编程语言 | Python 3.11+ |
| 任务编排 | LangGraph / 自研图引擎 |
| 知识检索 | 向量数据库（Milvus / Qdrant）+ 混合检索 |
| LLM 接入 | OpenAI API / DeepSeek API / 本地模型 |
| 工作流引擎 | 基于有向无环图（DAG）的自研引擎 |
| API 框架 | FastAPI |
| 可观测性 | OpenTelemetry + Prometheus + Grafana |
| 前端框架 | React + TypeScript + Vite |
| 可视化引擎 | React Flow / Cytoscape.js（工作流画布） |
| UI 组件库 | Ant Design / shadcn/ui |
| 状态管理 | Zustand / Redux Toolkit |
| 图表与监控 | ECharts / D3.js（监控看板） |
| 存储 | PostgreSQL + Redis |
